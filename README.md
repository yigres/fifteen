### Installation  
```sh
$ git clone git@github.com:yigres/fifteen.git
$ cd fifteen
$ make install
```  
### Usage
```sh
$ npm start
```  
### Play
Move:
* ArroLeft
* ArrowUp
* ArrowRight
* ArrowDown  
  
Only do it!!!
